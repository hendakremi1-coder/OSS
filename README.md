# OSS Business Case Dashboard
## Expo 2030 Riyadh Company | Confidential

Interactive financial and governance dashboard covering:
- **GAP 1** — Financial Model with live variable inputs and auto-recalculating projections
- **GAP 2** — Delivery Model scoring matrix with adjustable weights
- **GAP 3** — Governance framework, RACI matrix, and KPIs

### Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and import the repository
3. Vercel auto-detects `index.html` as a static site — no build settings needed
4. Click Deploy

### Local preview

Open `index.html` directly in any browser. No server required.

### Variables (GAP 1)

All inputs are editable via sliders on the Financial Model tab:
- Total Pavilions
- Platform Adoption Rate
- Avg. Transaction Volume per Pavilion
- Transaction Fee Rate
- Registered Suppliers
- Avg. Supplier Subscription Fee
- CapEx
- Annual OpEx
- ERC Revenue Share (PPP Dynamic)

Every figure in the projection table, scenario analysis, KPI cards, and bar chart recalculates instantly when any slider is moved.
